화학 반응식에서 알 수 있는 사실 (p.24~25)
✅ 완성된 화학 반응식
CH₄ + 2O₂ → 2H₂O + CO₂
1

반응물과 생성물

화살표 왼쪽은 반응물,
오른쪽은 생성물

반응물: CH₄, O₂
생성물: H₂O, CO₂
2

원자의 종류와 개수

반응 전후 원자의 종류·수가 변하지 않음

C: 1개 → 1개 ✓
H: 4개 → 4개 ✓
O: 4개 → 4개 ✓
3

계수의 의미 (입자 수 비)

계수 = 입자 수의 비
반응·생성 양을 알 수 있음

CH₄ : O₂ : H₂O : CO₂
= 1 : 2 : 2 : 1
📐 계수비 자세히 보기
CH₄ + 2O₂ → 2H₂O + CO₂
메테인 1분자 + 산소 2분자 → 물 2분자 + 이산화탄소 1분자
계수비 CH₄ : O₂ : H₂O : CO₂ = 1 : 2 : 2 : 1
💡 메테인 분자 2개가 연소할 때 필요한 산소 분자는?
→ 계수비 1 : 2 이므로 산소 분자 4개
→ 생성되는 물 분자는 4개, 이산화탄소는 2개
🧪 스스로 확인하기 (p.25)

일산화 탄소(CO)가 연소하면 이산화 탄소(CO₂)가 생성된다.
이 반응을 화학 반응식으로 나타내면? 2CO + O₂ → 2CO₂
